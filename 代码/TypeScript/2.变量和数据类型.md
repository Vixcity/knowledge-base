变量的使用分为两步
1. 声明变量并且指定变量类型
2. 给变量赋值

```typescript
let age: number;
age = 18;
console.log(age)

// 初始化变量
let sex: string = '男'
```

1. 基本数据类型 => 5个

	2. string
	3. boolean
	4. undefined
	5. null
2. 复杂数据类型 => 两个
	1. object
	2. array

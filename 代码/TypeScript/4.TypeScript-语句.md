## 循环语句
```ts
// 循环语句  
// 重复做某件事情  
for ( let i: number = 0; i < 3; i++) {  
 console.log('北冥有鱼，其名为鲲，鲲之大，一锅装不下')  
}  
  
// 循环语句可以帮我们重复的做某件事情

// 计算 1-100 之间所有整数的和  
let sum: number = 0;  // 这里需要一个默认赋值为0，否则会报错
for(let i: number = 1; i <= 100; i++){  
 sum += i  
}  
console.log(sum);
```

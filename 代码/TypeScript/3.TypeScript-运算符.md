## 算数运算符
==> +(加) -(减) \*(乘) /(除)

```ts
console.log(1+2) // 3 加  
console.log(2-1) // 1 减  
console.log(2*2) // 4 乘  
console.log(4/2) // 2 除
```

+号也是连接符号

```ts
console.log(1+2) // 3  
console.log(1+'2') // 12  
console.log('1'+2) // 12
```

其它运算符不可以与字符串一同使用

但是可以将字符串类型转换成数字类型

```ts
// 可以这样使用，但是字符串必须是数字
let age: number = +'18'
```

赋值运算符，将右边的值赋值给左边的变量
 ==> +=,-=,\*=,/=
 ==> 加等，减等，乘等，除等
